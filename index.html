一站式构建区块链网络与服务

Hyperledger

是面向企业的分布式区块链账本平台，具有高度的灵活性、模块化和可扩展性能。7Qiao提供Hyperledger Fabric网络的快速构建与按需配置，并为开发者提供区块浏览器、链码管理、API调用等功能。

IPFS

IPFS（InterPlanetary File System，星际文件系统）是永久的、去中心化保存和共享文件的方法，这是一种内容可寻址、版本化、点对点超媒体的分布式协议。7Qiao提供了IPFS网络的接入节点，并支持通过该节点完成文件的上传和查询等功能。

Ethereum

以太坊(Ethereum)是一个基于智能合约的分布式应用平台。7Qiao提供以太坊测试网络的快速接入功能，并提供基于Remix的IDE，开发者可以在平台内上传智能合约、在线编辑，并完成编译、部署和调试

qiqiao hardware

七巧数据终端（7Qiao Cloud Nodes）内置CNFS云端服务组件，可高效完成分布式、点对点互联组网以提供云级服务。此外，七巧数据终端以开箱即用的形式部署有若干常用本地服务，并支持以应用商店形式扩充可用服务。

Oracle

Oracle服务提供包括随机数、天气等多个可信的外部数据接入服务，数据使用方可以通过Oracle服务生成并保存数据证明文件，同时，Oracle服务也提供证明文件的验证工具，用以验证数据信息的可信程度。

形式化验证

7Qiao与CertiK、Codefine等国际领先的形式化验证公司合作，提供最有竞争力的规模化验证服务来保证智能合约和区块链系统的安全性。
